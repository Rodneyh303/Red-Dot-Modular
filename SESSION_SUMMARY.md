🎵 SYSTEM SUMMARY
═════════════════

Red Dot Modular is a complete polyphonic stochastic sequencer ecosystem:

MONSOON (Voice 1, mono sequencer) →
  → STRAITS EAST (Voices 2-8, add 7) →
  → STRAITS WEST (Voices 9-16, add 8)

DNA CONTROL OPTIONS:
  • Sands Mono: Global DNA (all 16 voices)
  • East/West Sands: Per-voice DNA (2-8 or 9-16)
  • Sands Macro: Global interpolation (blend all voices)

SPREAD MODES:
  • Poly Average: Voices pull toward each other
  • Mono Source: Voices follow main sequencer

MODULATION:
  • Interchange: Modulates Monsoon's pitch controls globally
