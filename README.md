# MeloDicer VCV Rack Plugin

A VCV Rack 2 module inspired by Vermona **meloDICER** — stochastic melody & rhythm with edit-menu features.

## Build

1. Install the VCV Rack SDK.
2. Put this folder inside your Rack `plugins/` directory.
3. Run `make RACK_DIR=/path/to/Rack-SDK` from this plugin’s directory.
4. Launch Rack; find **Red Dot Modular → MeloDicer**.

## Notes

This package is a compilation-friendly scaffold. Replace or expand DSP/UI bits with your final implementations as needed.
